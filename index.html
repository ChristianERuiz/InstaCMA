Free CMA generator
Try it now — no signup needed.

Follow these steps every time you run a CMA. Takes about 3 minutes from Paragon to finished report.

1
Set up your Paragon search by zip code

In Paragon, go to Search → Residential. Set the following criteria:

  • Status: SLD (Sold)
  • Zip Code: Enter the subject property's zip code — this pulls the right market automatically without guessing at radius or area codes
  • General Date (Begin): Set to 1 year back from today — leave End blank
  • Leave all other fields blank — let the AI filter the rest
  • Not enough comps? (fewer than 20 sold): Add a neighboring zip code to the search and re-export. The AI will flag any comps from a different market and exclude them automatically
2
Add the required columns via Customize

After running the search, click Customize in the results toolbar. Make sure these columns are visible before exporting:

3
Export as CSV

Click Export → CSV from the results toolbar. Save the file to your desktop. This is the file you upload in Step 2 of the CMA tool.

4
Fill in the subject property

Go to the Subject Property tab. The most important field is Total Finished Sq Ft — always use the confirmed actual sq ft, not the public record (they often differ). Add condition, garage count, any recent upgrades, and the effective age if the home has been renovated.

5
Generate and download your report

Upload your CSV, add your branding, then hit Generate CMA Report. The AI will analyze all comps, flag outliers, apply adjustments, and return a full report. Download the branded PDF and you're ready for your listing appointment.

💡 Pro tip — why zip code beats radius

Zip codes match how buyers actually think about neighborhoods. A radius circle can accidentally pull in homes from a completely different market — across a highway, in a different price tier, or in a rural vs. suburban setting. A zip code search keeps you in the right market automatically. If you get fewer than 20 comps, just add a neighboring zip and the AI will sort out which ones belong.
